Integrantes do grupo: THALLES MATUZAEL DANTAS ANDRADE, THIAGO DE SOUZA CONRADO DOS PASSOS, JEREMIAS FERREIRA SANTANA.

Tema central: Autoria Web – processo de criação, desenvolvimento e publicação de conteúdo para a web, com foco nas três tecnologias essenciais: HTML (estrutura), CSS (estilização) e JavaScript (interatividade).
Objetivos: Ensinar os fundamentos do desenvolvimento web de forma prática, com exemplos interativos e demonstração de oportunidades profissionais na área.

TECNOLOGIAS UTILIZADAS

HTML5: estrutura semântica, elementos modernos, acessibilidade.
CSS3: variáveis, flexbox, grid, responsividade, animações.
JavaScript (ES6+): manipulação do DOM, eventos, APIs do navegador.
Bibliotecas externas: Font Awesome (ícones), Google Fonts (tipografia), Unsplash (imagens).
APIs do navegador: rolagem suave, clipboard, Intersection Observer.

ESTRUTURA DO SITE

Cabeçalho com navegação responsiva.
Seção principal (Hero) com chamada para ação.
Seções dedicadas a: Autoria Web, HTML, CSS, JavaScript e Conclusão.
Rodapé com links e informações técnicas.

FUNCIONALIDADES PRINCIPAIS

Design responsivo (mobile-first).
Carrossel interativo com demonstrações de JavaScript.
Comparação visual “CSS vs. sem CSS”.
Botão para copiar códigos.
Animações durante a rolagem.
Navegação suave entre seções.

PADRÕES DE DESENVOLVIMENTO

Código limpo e organizado.
Sistema de design com cores e tipografia consistentes.
CSS estruturado com variáveis e media queries.

ASPECTOS EDUCACIONAIS

Abordagem “aprenda fazendo” com exemplos práticos.
Explicações claras e código comentado.
Progressão lógica: HTML → CSS → JavaScript.

PONTOS FORTES

Design moderno e responsivo.
Conteúdo educacional bem estruturado.
Implementação técnica de qualidade.
Interatividade que engaja o usuário.
Código documentado e reutilizável.

APLICABILIDADE

Material de apoio para cursos técnicos.
Portfólio para desenvolvedores iniciantes.
Base para projetos futuros.
Referência educacional em autoria web.
